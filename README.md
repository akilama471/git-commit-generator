# 🚀 git-commit-generator

## 📝 Description

`git-commit-generator` is a Python-based command-line tool designed to assist developers in generating meaningful and well-formatted Git commit messages. Leveraging AI capabilities, this tool aims to streamline the commit process by suggesting relevant commit messages based on code changes or provided context.

## ✨ Features

*   **AI-Powered Commit Generation:** Generates commit messages using AI models, suggesting appropriate prefixes (feat, fix, docs, etc.) and concise descriptions.
*   **Command-Line Interface (CLI):** Provides an intuitive CLI for easy integration into your Git workflow.
*   **Customizable Configuration:** Allows users to configure AI model parameters and commit message formatting.
*   **Dependency Management:** Utilizes `setup.py` for easy installation and management of project dependencies.
*   **Extensible Core Modules:** Designed with modularity, allowing for future enhancements and integrations.

## 🛠️ Installation

### Prerequisites

*   Python 3.7+
*   `pip` package installer

### Installation Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/git-commit-generator.git
    cd git-commit-generator
    ```

2.  **Install the package:**
    ```bash
    pip install .
    ```
    Alternatively, for development purposes, you can install it in editable mode:
    ```bash
    pip install -e .
    ```

## 💡 Usage

### Basic Usage

Once installed, you can use the `git-commit-ai` command from your terminal.

*   **Generate a commit message based on staged changes:**
    ```bash
    git commit-ai
    ```
    This command will analyze your staged changes and propose a commit message. You will be prompted to confirm or edit the message.

*   **Generate a commit message with a custom prompt:**
    ```bash
    git commit-ai --prompt "Refactor user authentication module"
    ```
    This allows you to guide the AI with specific instructions.

### Advanced Options

Refer to the CLI help for a full list of options:
```bash
git commit-ai --help
```
